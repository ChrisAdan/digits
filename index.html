Thanks for playing with this calculator. I made it as part of The Odin Project's final Foundations exercise to put into practice the HTML, CSS, and JavaScript learned thus far.

It was a fun experience making this and I ran into lots of pitfalls. In the JS, I wanted to better organize my code according to this idea of separation of concerns using an MVC architecture. Doing so introduced plenty of bugs to chase down in order to get the math to properly evaluate, but I also found that it made the code easier to parse by simplifying each additional component according to it's narrow function, and using the paradigm of imports to interact between modules. Also a fun challenge was bringing in the keyboard support, especially properly handling special characters such as +/= or 6/^.


The CSS posed another interesting wrinkle. Understanding that the Grid exists to simplify this type of layout for the calculator buttons, the project requirements were explicit that Flexbox was to be used instead. I still have not fully perfected the 'exact equal dimensions' I was seeking for my buttons (looking at you, modulus). However, one should never let perfect be the enemy of good enough.


For all other learners, keep up the journey and don't give up! Every new thing that we build, we learn more and continue to grow. I appreciate your time.
